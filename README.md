### Debian/Ubuntu
Install some dependencies:
```bash
sudo apt update && sudo apt install -y git wget rsync zsh vim curl
```

## Install 
```bash
git clone https://github.com/mab-eric313/dotfiles.git
cd dotfiles
./install.sh
```
